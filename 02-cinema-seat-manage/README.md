# Cinema Seat Manager

A Java console project created to practice two-dimensional arrays by simulating the seat management system of a movie theater.

## Project Description

This program represents a movie theater using a two-dimensional integer array.

Each position in the matrix represents one seat:

- `0` means the seat is available.
- `1` means the seat is occupied.

The user can view the complete theater, reserve seats, cancel reservations, check the status of a specific seat, and count the available and occupied seats.

## Main Features

The program allows the user to:

1. View the complete seating arrangement.
2. Reserve a seat by entering its row and column.
3. Cancel an existing reservation.
4. Check whether a specific seat is available or occupied.
5. Count the available and occupied seats.
6. Exit the program.
